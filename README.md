# Paws Shop - Pet Supplies Management System

Paws Shop is a Java Swing-based desktop application designed for managing pet store inventory, supplies, cashiers, and checkout/billing workflows. 

## 🌟 Key Features

* **Secure Cashier Login:** Built-in validation system for employee authentication.
* **Product & Supplies Inventory:**
  * View current supplies list.
  * Search supplies dynamically by tags, categories, or names.
  * Add new pet products and stock items to the system.
* **Cashier Registration:** Add and manage cashier credentials.
* **User-friendly Desktop GUI:** Styled using Java Swing layout panels (`.form` configuration templates generated by NetBeans).

## 🛠️ Technology Stack

* **Programming Language:** Java 8+
* **Framework:** Java Swing / AWT (NetBeans GUI Form Designer)
* **Build System:** Ant (`build.xml`)

## 🚀 Getting Started

### Prerequisites
* **Java Development Kit (JDK)** version 8 or higher.
* **NetBeans IDE** (Recommended for viewing and editing `.form` files) or any other Java-compatible IDE (IntelliJ IDEA, Eclipse).

### Running the Application

#### Using NetBeans
1. Open NetBeans IDE.
2. Select **File > Open Project** and choose the `paws_shop` directory.
3. Clean and build the project.
4. Run [Start.java](src/com/pawsshop/pawsshop/Start.java).

#### Using Command Line
Navigate to the project directory and build/run with Ant:
```bash
ant compile
ant run
```
